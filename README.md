## 🌐 **ویژگی‌ها و امکانات**
✔️ **اسکن آی‌پی (IPv4 & IPv6)**  
✔️ **دریافت کانفیگ WireGuard**  
✔️ **ایجاد لینک سابسکریپشن**  
✔️ **پشتیبانی از V2ray و MahsaNG**  
✔️ **پشتیبانی از Sing-box و Hiddify**  
✔️ **ذخیره آی‌پی‌ها در فایل CSV**  
✔️ **دریافت کانفیگ  Warp on Warp**
