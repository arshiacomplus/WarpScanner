## 🌐**機能と機能**

✔️**IPスキャン（IPv4＆IPv6）**  
✔️**دریافت کانفیگ WireGuard**  
✔️**サブカーニアルリンクの作成**  
✔️**v2rayとMahsangをサポートします**  
✔️**Sing-BoxとHiddifyをサポートします**  
✔️**IPSをCSVファイルに保存します**  
✔️**ワープでワープ構成を取得します**
