## 🌐**功能和功能**

✔️**IP扫描（IPv4和IPv6）**  
✔️**获取线条配置**  
✔️**创建一个亚概念链接**  
✔️**支持V2RAR和MAHSANG**  
✔️**支持Sing-Box和Hiddify**  
✔️**将IP保存在CSV文件中**  
✔️**在翘曲上获取经线配置**
